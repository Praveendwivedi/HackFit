# HackFit

## Setup Guidelines:

- # Only for Windows:
  - # in powershell
    - Install chocolatey from [here](https://chocolatey.org/install).

    - Then, ```choco install make```.
- clone the repository
- in terminal run following commands:
  - ```cd HackFit```
  - ```make```
  - for Windows:
    - ```. HFenv\Scripts\activate```
  - for linux:
    - ```. ./HFenv/bin/activate```
  - finally:
    - ```make run```
